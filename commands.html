Comandos de DP

startdp - Inicia uma partida de Dungeons & Pokémon.

defplayers [jogadores] - Define os jogadores da partida de DP.

defclass [jogador], [classe] - Define as classes dos jogadores.

confirmclass - Confirma as classes dos jogadores.

act [jogador], [habilidade], [alvos] - Registra uma ação dentro da partida.

actsconfirm - Confirma todas as ações que registrou no round.

cancelact - Cancela a última ação registrada.

spirit [jogador spirit], [jogador possuído] - Passiva do Spirit.

trapper [jogador trapper], [armadilha] - Passiva do Trapper.

makehost [usuário] - Faz um usuário da sala ser também host da partida.

finishdp - Acaba com a partida.

Comandos Mega Quiz

mq [sala], [questão]- Registra uma pergunta em uma sala.

cancelquestion - Cancela a pergunta atual.

add [alternativa] - Registra uma alternativa na pergunta.

danswer [alternativa] - Define a resposta correta.

send - Manda a pergunta feita na sala.

showquestion - Mostra como está a questão, com a pergunta e as alternativas

respond [sala], [jogador-host], [alternativa] - Responde uma pergunta, feito automaticamente após clicar numa alternativa.

definetimer [sala], [tempo] - Define o tempo de resposta após uma questão ser mandada.

Comandos Gerais

commands - Mostra este HTML.

addpoints [sala], [usuário], [número de pontos] - Adiciona, numa sala, pontos a uma pessoa.

removepoints [sala], [usuário], [número de pontos] - Remove pontos de uma pessoa numa sala.

clearpoints [sala]- Limpa a leaderboard de uma sala.

leaderboard [sala]- Mostra a pontuação de uma sala.

timer [tempo], [min/sec] - Cronômetro em minutos/segundos.
